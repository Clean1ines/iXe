### **Задача 6. Обновление Makefile и валидация DevEx**

**1. Команды терминала для получения контекста:**

```bash
cat Makefile
```

**2. Блок для вставки полученного контекста:**

(Пользователь вставляет *содержимое Makefile*)

**3. Само задание:**

**Цель:** Упростить локальную разработку с `common/`.

**Действия:**
- Добавить в `Makefile`:
  ```makefile
  install-common:
  	pip install -e common/
  ```
- Добавить `install-common` как зависимость для `install` или `dev` цели, если такая существует

**Acceptance Criteria:**
- Команда `make install-common` работает
- После `make install-common` все импорты `common.*` разрешаются в REPL и при запуске сервисов
