**3. Само задание:**

**Цель:** Заменить все старые импорты на `from common.*`, затем удалить исходные файлы.

**Действия:**
- Заменить импорты:
  - `from models.X import Y` → `from common.models.X import Y`
  - `from utils.model_adapter import ...` → `from common.utils.model_adapter import ...`
  - Аналогично для `processors`, `services.specification`, `config`
- Убедиться, что импорты в `common/` сами себя не ломают (например, `common/utils/model_adapter.py` не импортирует из `common/models` с ошибками)
- Удалить старые файлы из `models/`, `utils/`, `processors/`, `services/`, `config/` (только после успешного запуска тестов)

**Acceptance Criteria:**
- Команда `grep -r "from models\.\|from utils\.model_adapter\|from processors\." --include="*.py" . | grep -v common/ | wc -l` возвращает 0
- Все тесты проходят: `pytest`
- Приложение запускается: `uvicorn web_api.main:app --reload`
- Старые файлы удалены
