### **Задача 5. Интеграция в DevOps и безопасность**

**1. Команды терминала для получения контекста:**

```bash
ls Dockerfile*
grep -A5 -B5 "COPY.*\." Dockerfile.web 2>/dev/null || echo "Dockerfile not found"
```

**2. Блок для вставки полученного контекста:**

(Пользователь вставляет *содержимое Dockerfile.web* и список Docker-файлов)

**3. Само задание:**

**Цель:** Обеспечить корректную сборку образов и базовую безопасность `common/`.

**Действия:**
- Для каждого `Dockerfile.*` добавить:
  ```dockerfile
  COPY common/ ./common/
  RUN pip install -e ./common/
  ```
- Выполнить `bandit -r common/` и устранить HIGH/CRITICAL уязвимости (если есть)
- Убедиться, что `common/` не содержит секретов: `grep -r "SECRET\|KEY\|PASSWORD" common/ || echo "clean"`

**Acceptance Criteria:**
- Все `Dockerfile.*` обновлены
- `docker build -f Dockerfile.web .` завершается успешно
- `bandit -r common/` не выдает HIGH/CRITICAL
- Поиск секретов возвращает пустой результат
