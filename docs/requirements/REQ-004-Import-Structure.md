# Требование: Структура импортов

**ID:** REQ-004
**Тип:** Архитектурное
**Требование:** Структура импортов в проекте должна отражать архитектурное разделение. Импорты из `common` должны быть явными и следовать шаблону `from common.<module>.<submodule> import ...`.
**Обоснование:** Поддержание ясной архитектуры, облегчение понимания зависимостей компонентов.
**Критерии приёмки:**
- Все файлы, использующие общие компоненты, импортируют их из `common`.
- Импорты из `common` используют полный путь (например, `from common.models.problem_schema import Problem`).
- Не допускается импорт напрямую из старых директорий (`models`, `utils` и т.д.), если компонент был перенесён в `common`.
**Связанные ADR:** `docs/adr/002-common-library-structure.md`
