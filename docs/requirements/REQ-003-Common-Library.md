# Требование: Использование общей библиотеки

**ID:** REQ-003
**Тип:** Архитектурное
**Требование:** Все компоненты, являющиеся общими для разных частей проекта (скрапер, API, сервисы, скрипты), должны быть реализованы и находиться в `common/`. Все импорты этих компонентов должны происходить из `common.*`.
**Обоснование:** Устранение дублирования кода, обеспечение согласованности данных и логики, упрощение сопровождения и рефакторинга.
**Критерии приёмки:**
- Все общие модели данных (Pydantic/SQLAlchemy) находятся в `common/models/`.
- Все общие stateless-утилиты находятся в `common/utils/`.
- Все общие процессоры данных находятся в `common/processors/`.
- Все общие сервисы бизнес-логики находятся в `common/services/`.
- В проекте отсутствуют дублирующие реализации компонентов, уже находящихся в `common`.
- Все файлы в проекте (кроме `common/`) используют импорты вида `from common.*` для доступа к общим компонентам.
